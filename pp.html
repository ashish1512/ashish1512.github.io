Intelligent Hearing Aid for the Hearing Impaired
Even with technological advances in every field, the world is still not accessible to people with a hearing disability. Built an E2E prototype that would help people with a hearing disability in their daily lives. Trained a neural network to recognize different sounds such as a dog barking, a car horn or a ringing doorbell using stochastic gradient descent and backpropagation. Connected this system, written in Python using Tensorflow, to a wrist band which had unique vibration patterns to represent each sound using Arduino.
Content Check - Sentiment Analyzer
With growing privacy concerns on social media, built a browser extension to check the content of a post a user is going to publish on any social media platform. It tells the user whether the post contains any objectionable content that would otherwise hurt the sentiment of a group of people.
It also provides the user with a handy summary of other posts talking about similar topics on these social media platforms. It classifies the post’s sentiment as positive, neutral or negative and leverages the MeaningCloud API to perform topic extraction from the text to analyze global or location specific opinions. Backend is written as a REST API in Python using Flask and front end is in Javascript.
MedGuide
To address the gap between doctors and patients, developed a website where users can upload medical reports and gain contextual information about them. Designed an algorithm which performed natural language processing on the report to extract medical terms and provide basic definitions for each. Extended the web app to present the user with health recommendations using APIMedic for certain Vitamin deficiencies. (link)
Number Recognizer

HighQ - News Platform
Independently developed an E2E news platform using ASP.NET and ReactJS. Users could view latest news articles summarized in less than 80 words, search through those articles and view them in 8 languages. Publishers can take advantage of the platform to connect with users and write popular news content. It received over 20000 hits within the first 2 months. (link)
Photo Tagger
Built a multi-class classifier in Python using SKLearn to train a convolutional neural network to tag photos into various categories such as flowers, cars, buildings or faces as a part of TagMe!, an inter-college competition at IISC Bangalore; trained on a set of 5000 images with an accuracy of 79%.
